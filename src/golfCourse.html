Golf Course Website

  1. Home
  2. Projects
  3. Golf Course Website

Overview

This project was created for a class assignment in 2025. To complete this project, I had to make a Figma prototype. Then, code this project using the prototype as a template. I had free rein to choose the contents of this website. I chose to make a golf course website because I work at a golf course and have experience with the needs of golfers.

Process

For this project, I used HTML, CSS, Bootstrap, and Figma. This website is responsive for mobile, tablet and desktop screen sizes. It was important to me to ensure that all websites tasks were possible and optimized. In total my website had 6 main pages which are: the home page, the tee times page, the shop page, the association page, the lessons page, and the events page.

Takeaways

I learned a lot from this project. One of the main aspects I learned is how to build a pagination. For the tee times page, I wanted the user to be able to change which day they were looking at. I also used many images for this project. I learned how to make images more responsive and fit properly in their containers through this project.
