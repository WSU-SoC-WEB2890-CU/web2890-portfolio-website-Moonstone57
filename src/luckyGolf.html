  1. Home

Overview

This project is being created for a class assignment in 2026. However, lucky.golf is an actual business website. The purpose of the website is to be the online pro shop for Crane Field and Remuda golf courses. Through this website, users can buy golf products, sign up for an event, join an association, and more. For the class requirements, I was supposed to find a business that would let me redesign their website.

Process

For this project, I used HTML, SASS, JavaScript, and Bootstrap. This website is responsive for mobile, tablet and desktop screen sizes. This project was personal to me as I currently work at Crane Field Golf Course. I have been able to use my knowledge of their customer base to know what features to prioritize. I got design feedback from the client and other programmers. I used their advice to hone in the functionality of the website.

Takeaways

The goal of this redesign was to make the website information more concise and simpler to navigate. There are also a lot of confusing links and sections. This website had a challenging color palette and I got a lot of feedback on how to implement the colors in a effective way. I have learned how to take advantage of vibrant colors and not overwhelm the website with them. I also learned how important feedback is to create a better end product.
