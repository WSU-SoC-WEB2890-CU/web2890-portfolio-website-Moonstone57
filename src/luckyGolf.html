Overview

This project is being created for a class assignment in 2026. However, lucky.golf is an actual business website. The purpose of the website is to be the online pro shop for Crane Field and Remuda golf courses. Through this website users can buy golf products, sign up for an event, join an association, and more. For the class requirements I was supposed to find a business that would let me redesign their website.

Process

For this project I am using HTML, SASS, JavaScript, and Bootstrap. This website is responsive for mobile, tablet and desktop screen sizes.

Takeaways

The goal of this redesign is to make the website information more concise. There are also a lot of confusing links and sections. The goal is to make the website have less information overload and be simpler to navigate.
