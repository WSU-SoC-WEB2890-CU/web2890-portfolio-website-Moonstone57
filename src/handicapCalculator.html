  1. Home

Overview

I created the Golf Handicap Calculator for a class assignment in 2025. For this assignment, I needed to create a webpage that would prioritize Javascript in a web application. I decided to create a golf handicap calculator. This website allows a user to input their score either through hole by hole or through total scores. The website then takes their score to calculate the difference of their score and par. Then it displays the handicap to the users.

Process

This website utilizes HTML, CSS, and JavaScript. Another important aspect of the website, that is created through Javascript, is saving the handicap information through local storage. This means that even if the user reloads the page they will not lose their already calculated handicap. It was also important that only the specific elements a user needs are shown on the screen.

Takeaways

This was my first JavaScript project. I learned a lot through this project. I learned how to apply link JavaScript to HTML and CSS. I also gained experience making a website’s form inputs truly functional. I found it quite challenging at first to figure out how to show and hide elements with JavaScript. This project gave me a greater insight on how to apply JavaScript in an effective way.
