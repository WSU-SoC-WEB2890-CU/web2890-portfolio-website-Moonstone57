Doorway to Fantasy Photoshop Image Website

  1. Home

Overview

This project was created as a class assignment in 2025. For this assignment, I had to use Adobe Photoshop to create a conceptual image. The purpose of this assignment was to represent a story or emotion using one image. For this project, I decided to depict a girl sitting on a hospital bed looking through a doorway into a fantasy world.

Process

For this project, I drew several sketches to solidify how I wanted to design my image. I then used Adobe photoshop to create the final image. The idea behind this project came from an experience I had in October 2025. My appendix ruptured in October and I had to be hospitalized for a week. I was extremely sick during this time and the entire time I was there I was picturing where I would rather be. As I went to create this project, I found an image of a girl on a hospital bed and I thought it would work perfectly.

Takeaways

I learned a great deal about image editing through this project. I feel this project taught me how to effectively utilize adjustment layers and color enhancements. I wanted the fantasy world to feel a bit unrealistic. It was meant to represent the idea of escaping reality. I wanted the viewer to be drawn toward first the butterfly and then the rest of the fantasy world.
