Overview

I created this webpage for a class assignment in 2026. For this assignment I was given the contents of the webpage by my professor. The purpose of this webpage is to let users sell their car to the company “CarDeal”. I arranged the content, such as the text, images, and logo, to be easy to understand and designed well. This webpage had many sections such as testimonials, recent purchases and a FAQ.

Process

For this project, I utilized HTML, SASS, and Bootstrap. I prioritized having a responsive webpage. The webpage can adjust to mobile, tablet, and desktop screen sizes. For this webpage I used premade Bootstrap components. However, one of the requirements for the webpage is that I edit the code and make the Bootstrap components my own.

Takeaways

A concept I learned from this project is how to make responsive forms. I have a form in my hero image. I found that initially to be challenging, but I feel I could make a responsive form more easily in the future.
